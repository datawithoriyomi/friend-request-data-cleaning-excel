# Friend Request Data Cleaning Using Excel

## Project Overview
This project focuses on cleaning and structuring historical friend request data collected between 2015 and 2023.

The raw dataset was provided in an unstructured format and required transformation into a clean, usable structure.

## Objective
- Clean and organize the data into Name, Date, and Time columns
- Improve data consistency and readability
- Prepare the dataset for analysis or record management

## Tools Used
- Microsoft Excel

## Approach
- Extracted and separated combined values
- Standardized date and time formats
- Structured the dataset for easy filtering and reporting

## Outcome
The final dataset is clean, structured, and analysis-ready.

## Skills Demonstrated
Data Cleaning • Data Structuring • Excel • Attention to Detail
